ttt(6)

NAME
     ttt - tic tac toe

SYNOPSIS
     ttt

EXAMPLES

     ttt                 # Start the game

DESCRIPTION

     This program allows the user to engage in a game of tic tac toe  (noughts
     and  crosses)  with  the  computer.   The  program  uses  the  alpha-beta
     algorithm, so the user had better be sharp.
